Visualizing how the new world of AI bots interact and compete with each other

Altered State Machine is a platform that lets you build, train and trade AI agents using non-fungible tokens (NFTs) for gaming and financial purposes. The protocol enables anyone to own an AI agent and the machine learning models that define what it learns over time. Owners pair an NFT intelligence agent (a Brain) with NFT artwork (a Form) to make their Agent in Metaverse.

For the collaboration with ASM, we created the entire website, including brand identity, animation, 2D and 3D illustrations dripping with sci-fi and futuristic vibes. We used tools like TypeScript, EJS, BodyMovin, Lottie, and GreenSock.

Building a brand identity through dynamic visuals

Our goal is to visualize the idea of premium NFT by bringing together brand identity with logo, color palette, and fonts. We used a visual metaphor of futuristic cosmic objects with the metallic glow to the smallest of them. The target audience of this project is investors, so in the visuals, we emphasize the value and integrality of these objects and give their presence a subtle weightiness.
